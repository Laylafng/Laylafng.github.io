MongoDB安装使用指北自用

这段时间有高级软件工程这门课，其中个人实验作业需要用到MongoDB数据库，之前用的都是MySQL，确实没用过MongoDB，所以这里记录MongoDB的安装和一些使用的简单教程，主要是用来自己查找指令。

安装

下载mongodb

Download MongoDB Community Server | MongoDB

  1. 下载完之后就正常安装，注意自定义安装，就不要勾选那个可视化界面，不然下载完都不知道啥时候了





  2. 安装完成之后在文件夹下面建立新的数据库db和日志文件log
  3. 在log文件夹下建立mongodb.log
  4. 在MongoDB的bin文件同层建立mongod.config
  5. 内容需要自己设置
  6. 然后在管理员状态运行指令，需要到mongodb的bin文件夹下
    mongod –config “D:\softhome\mongod.config” –install –serviceName “MongoDB”
    成功后按win+r运行框中输入services.msc,检查MongoDB服务是否开启就行，还可以通过http://localhost:27017/测试
  7. 最后就是配置环境变量，把mongodb下面的bin文件夹添加到Path就可以啦
    然后因为我安装的是6.0版本，没有那个mongo的shell指令来，需要自己单独下载

下载mongodb shell

MongoDB Shell

安装zip文件之后直接解压缩，把bin文件下面的两个文件直接复制到mongodb的bin文件夹下面就可以
